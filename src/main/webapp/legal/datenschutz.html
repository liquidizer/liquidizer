Datenschutzerklärung

Alle vom Liquidizer erfassten Daten werden im Rahmen der gesetzlichen Vorschriften geschützt. Nachfolgend findest Du Informationen darüber, was erfasst wird.

Es werden nur explizit eingegebene Daten vom System erfasst. Dazu gehört der Nutzername, die abgegebenen Stimmgewichte, die hinterlassenen Kommentare und die im System gestellten Forderungen. Mit Ausnahme der gestellten Forderungen können alle Daten vom Benutzer selbstständig wieder gelöscht werden. Alte Stimmgewichte, Delegationen, Kommentare und Profileinstellungen werden sofort vom System gelöscht, sobald ein Nutzer diese ändert. Eine Historisierung findet nicht statt.

Die abgegebenen Stimmgewichte werden themenbezogen summiert und sind in ihrer Summe als historische Zeitreihe (Chart) abrufbar. Es erfolgt keine Speicherung von Daten, die Rückschlüsse auf die Identität oder das Pseudonym der an der Abstimmung beteiligten Nutzer oder deren Stimmgewichte zulässt.

Alle vom System erfassten Daten können von den Nutzern auf den entsprechenden Seiten abgerufen werden. Mit Ausnahme der optional angegebenen Email-Adresse sind diese öffentlich, d.h. auch anderen Nutzern zugänglich. Es werden keine verborgenen Nutzungsstatistiken angelegt.
